# PetMate

**문제**  
반려견 건강 데이터를 종이에 기록하거나 앱을 여러 개 써야 해서 관리가 불편함.

**해결**

- Next.js + Supabase로 통합 기록/조회 기능 구현
- AI 기반 자동 건강 리포트 제공

**성과**

- 초기 사용자 30명 → 주간 활성 사용자 100명 돌파
- 건강 기록 유지율 70% 이상
