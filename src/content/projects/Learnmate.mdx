# LearnMate

맞춤형 학습 관리 플랫폼으로, 학생들의 학습 진도와 성취도를 체계적으로 관리하고 교사-학생 간 소통을 강화하기 위해 제작했습니다.

---

## 🎯 문제 정의 (Problem)

- 학생별로 진도와 성적 데이터를 일관되게 관리하기 어려움
- 교사 ↔ 학생 간 소통 채널이 제한되어 과제 피드백이 지연됨
- 학부모는 자녀의 학습 현황을 실시간으로 확인하기 어려움

---

## 💡 해결 전략 (Solution)

1. **학습 관리 기능**

   - 학생별 진도/성적을 Supabase DB에 구조적으로 저장
   - 차트 기반 성취도 시각화 제공

2. **실시간 소통**

   - 교사 ↔ 학생 간 실시간 메시지 기능
   - 공지/피드백을 알림으로 즉시 전달

3. **반응형 디자인**

   - Tailwind CSS로 모바일·태블릿·데스크탑 최적화
   - 학부모도 쉽게 접근 가능하도록 UX 단순화

4. **보안 & 접근성**
   - Supabase Auth 기반 로그인/권한 제어
   - 교사/학생/학부모 역할(role)에 따른 화면 차별화

---

## 🚀 결과 (Result)

- 시험 준비반 학생 20명 대상으로 **파일럿 운영**
  - 과제 제출 기한 준수율 **+35% 증가**
  - 학습 진도 관리 효율 **40% 향상**
- 교사의 관리 시간이 줄어들어 **학생 피드백 속도 2배 개선**

---

## 🔍 회고 (Retrospective)

- 장점:

  - 기술 스택(Next.js + Supabase) 선택으로 개발 속도와 확장성이 뛰어남
  - 사용자별 요구(학생/교사/학부모)를 분리한 UX 설계가 효과적

- 아쉬운 점:

  - 오프라인 연계 기능(출결 관리, 학부모 상담 기록)이 아직 부족
  - 채팅 UI/UX 개선 필요 (모바일에서 답장 기능 추가 예정)

- 다음 단계:
  - **AI 추천 기능**: 학생별 성취도에 따른 개인 맞춤 문제 추천
  - **학부모 대시보드**: 주간 학습 리포트 이메일 자동 발송

---

## 🛠 기술 스택

- **Frontend**: Next.js 15, React, Tailwind CSS
- **Backend(DB/Auth)**: Supabase
- **배포**: Vercel
- **기타**: Chart.js (시각화), Zustand (상태 관리)
