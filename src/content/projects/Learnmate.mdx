# LearnMate

학습 목표 기반의 **AI 튜터**와 **코스 빌더**로, 학습 경로를 자동 생성하고 진행률을 시각화합니다.

## 핵심

- 사용자 레벨 진단 → 과제/퀴즈 자동 생성
- 코스 빌더: 모듈/레슨/퀴즈를 드래그&드롭으로 구성
- 프로그레스/리텐션 대시보드

## 기술 포인트

- Next.js App Router · 서버 컴포넌트
- Supabase 인증/DB, Edge Function(채점)
- 스트리밍 채팅 + 캐싱
