# <span className="text-shine">LearnMate</span>

**일정 관리와 학습 기록을 한 곳에서 정리**하도록 돕는 서비스입니다.  
학습 루틴을 자동 추천하고, 필요한 흐름만 빠르게 들어가도록 설계했어요.

## 주요 기능

- 개인 목표 기반 학습 플랜 자동 생성
- 진행 기록을 기반으로 학습 루프 관리
- 멘토 코멘트와 리마인더로 유지

## 기술 스택

- Next.js 15(App Router, MDX), Turbopack
- Tailwind CSS v4, Framer Motion
- Vercel 배포
