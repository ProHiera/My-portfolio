# <span className="text-shine">LearnMate</span>

**혼자 공부할 때도 끊기지 않도록** 일정, 피드백, 감정 기록을 한 화면에서 관리합니다.  
학습 루틴을 자동 제안하고, 오늘 할 일에만 집중하도록 인터페이스를 단순화했어요.

## 핵심 기능

- 개인 목표 기반 **학습 플랜 자동 생성**
- 진행률/기분 기록으로 **회고 루프** 구축
- 멘토 코멘트/리마인더로 **유지율** 향상

## 기술 스택

- Next.js 15(App Router, MDX), Turbopack
- Tailwind CSS v4, Framer Motion
- Vercel 배포
