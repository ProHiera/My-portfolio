# <span className="text-shine">BikeTown</span>

자전거 매장 운영에 맞춘 **브랜드/상거래/오프라인 예약** 경험을 한 곳에 모은 웹앱입니다.  
상품 탐색 → 픽업 예약 → 매장 방문까지의 여정을 최소 클릭으로 연결하는 데 집중했어요.

## 핵심 기능

- 상품/서비스 탐색과 **픽업 예약** 플로우 최적화
- **지도 연동**으로 매장/재고 확인
- 프로모션 섹션과 후기 위젯으로 **전환율** 개선

## 기술 스택

- Next.js 15(App Router, MDX), Turbopack
- Tailwind CSS v4, Framer Motion
- Vercel 배포
