# BikeTown

**문제**  
자전거 동호인들이 모이는 오프라인 몰에 온라인 지원 서비스가 전무함.

**해결**

- 의류/자전거 용품 온라인 판매 시스템 구현
- 카페·정비·식당 예약, 픽업 주문 기능 추가
- 지도 API를 활용한 자전거 길 정보 제공

**성과**

- 예약/픽업 기능으로 사용자 편의성 개선
- 지역 커뮤니티 활성화 (하루 방문자 200명 이상 가정)
