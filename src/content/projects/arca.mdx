# <span className="text-shine">ARCA</span>

ARCA는 **일정을 자동으로 정리하고 우선순위를 추천**해 주는 AI calendar app입니다.  
흩어진 일정 입력을 하나로 모으고, 중요한 일부터 처리할 수 있도록 흐름을 단순화했습니다.

## 주요 기능

- 자연어 일정 입력을 캘린더 이벤트로 자동 변환
- 일정 중요도/충돌 기반 우선순위 추천
- 반복 일정과 리마인더 자동 보정

## 기술 포인트

- AI 기반 일정 분류 및 추천 로직
- 모바일 중심 캘린더 UX
- 실시간 일정 동기화 확장 가능 구조
